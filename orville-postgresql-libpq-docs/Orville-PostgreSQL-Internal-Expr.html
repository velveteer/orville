Checks that the tuple constructed from the given columns in one of the tuples specified in the input list. It is up to the caller to ensure that all the tuples given have the same arity.

Checks that the tuple constructed from the given columns is NOT one of the tuples specified in the input list. It is up to the caller to ensure that all the tuples given have the same arity.
